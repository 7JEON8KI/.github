### 기획 배경
- 밀키트 시장 규모의 확대 & 밀키트 선호 연령층 확산
- 식품의 기본인 재료선택의 개인화

### 주요 기능
- 밀킷 추천 : 비슷한 연령대나 성별이 많이 구매한 상품 추천
- 개인 맞춤화 서비스 : 선호 재료/비선호 재료에 따른 개인 추천 서비스
- 이미지 검색 : 이미지 유사도 기반 이미지 검색 기능
